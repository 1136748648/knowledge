# FR-AUTH: 认证授权模块需求

## FR-AUTH-001: 本地用户登录
- **描述**: 用户通过用户名/密码登录系统，获取JWT令牌
- **参与者**: 所有用户
- **前置条件**: 用户已注册，系统已初始化
- **主成功场景**:
  1. 用户访问登录页
  2. 输入用户名和密码
  3. 系统验证凭证
  4. 返回JWT令牌和用户信息
- **扩展场景**:
  - 密码错误：显示错误提示，允许重试
  - 账号锁定：提示联系管理员
- **拟合标准**: 登录请求响应时间<1秒，密码错误3次后锁定5分钟
- **来源**: PRD用户故事#2
- **KANO类别**: 基本型
- **优先级**: P0
- **状态**: ✅ 已完成

## FR-AUTH-002: 内置管理员初始化
- **描述**: 系统未初始化时，使用内置管理员账号登录
- **参与者**: 系统管理员
- **前置条件**: 系统首次启动，无管理员账户
- **主成功场景**:
  1. 访问登录页，检测到未初始化状态
  2. 使用默认凭证登录
  3. 进入初始化向导
- **拟合标准**: 内置管理员账号`builtin-admin`可正常登录
- **来源**: PRD用户故事#1
- **KANO类别**: 基本型
- **优先级**: P0
- **状态**: ✅ 已完成

## FR-AUTH-003: 用户密码修改
- **描述**: 用户可修改自己的登录密码
- **参与者**: 所有用户
- **前置条件**: 用户已登录
- **主成功场景**:
  1. 进入个人设置页
  2. 输入旧密码和新密码
  3. 系统验证并更新密码
- **拟合标准**: 密码强度符合要求（8位以上，含大小写和数字）
- **来源**: PRD用户故事#2
- **KANO类别**: 期望型
- **优先级**: P2
- **状态**: ✅ 已完成

- **来源**: PRD用户故事#3
