# FR-WIKI: Wiki文档模块需求

## FR-WIKI-001: Wiki文档创建
- **描述**: 创建支持Markdown格式的Wiki页面
- **参与者**: 内容作者
- **前置条件**: 用户已登录，有创建权限
- **主成功场景**:
  1. 进入Wiki编辑页
  2. 输入标题和内容（Markdown）
  3. 设置敏感度级别
  4. 保存文档
- **扩展场景**:
  - 内容为空：提示需输入内容
  - 标题重复：提示修改标题
- **拟合标准**: 文档保存成功，自动创建版本记录
- **来源**: PRD用户故事#5
- **KANO类别**: 基本型
- **优先级**: P0
- **状态**: ✅ 已完成

## FR-WIKI-005: 文档敏感度分级

- **前置条件**: 创建或编辑文档
