# FR-NAV: 知识导航模块需求

## FR-NAV-001: 导航树CRUD
- **描述**: 创建、编辑、删除导航树节点
- **参与者**: 管理员
- **前置条件**: 用户已登录，有管理权限
- **主成功场景**:
  1. 进入导航管理页
  2. 创建/编辑/删除节点
  3. 保存更改
- **拟合标准**: 导航树结构正确，支持多级嵌套
- **来源**: PRD用户故事#11
- **KANO类别**: 期望型
- **优先级**: P1
- **状态**: ✅ 已完成

## FR-NAV-002: 内容关联
- **描述**: 将Wiki文档关联到导航节点
- **参与者**: 内容作者、管理员
- **前置条件**: 文档和导航节点都已存在
- **主成功场景**:
  1. 进入文档编辑页
  2. 选择关联的导航节点
  3. 保存关联关系
- **拟合标准**: 文档在导航树中正确显示，可通过导航访问
- **来源**: PRD用户故事#14
- **KANO类别**: 期望型
- **优先级**: P2
- **状态**: ✅ 已完成

## FR-NAV-003: 自动分类推荐
- **描述**: 文档提交时系统自动推荐导航类目
- **参与者**: 内容作者
- **前置条件**: 创建新文档
- **主成功场景**:
  1. 用户输入文档内容
  2. 系统分析内容并推荐类目
  3. 用户确认或选择其他类目
- **拟合标准**: 推荐准确率>80%
- **来源**: PRD用户故事#12
- **KANO类别**: 兴奋型
- **优先级**: P2
- **状态**: ✅ 已完成（Agent级别）
