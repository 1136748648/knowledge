## 下一步优先级建议

1. **补全 Phase 1 剩余**：用户管理 CRUD、Keycloak 集成、审计日志查询
2. **Phase 2 收尾**：Wiki 创建时集成分类+审核 Agent、定期重分类任务
3. **Phase 3 启动**：协调 Agent 跨库查询、QA 页面对接向量搜索
4. **测试覆盖**：为核心 Agent 和 Service 补充单元测试
